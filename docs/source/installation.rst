============
Installation
============

At the command line either via easy_install or pip:

.. code-block:: shell

    $ pip install hangman

.. code-block:: shell

    $ easy_install hangman

Or, if you have virtualenvwrapper installed:

.. code-block:: shell

    $ mkvirtualenv hangman
    $ pip install hangman

**Uninstall**

.. code-block:: shell

    $ pip uninstall hangman
